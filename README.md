# Ecoloop Back-End

This repository contains the backend of the **Ecoloop's 2nd Stage: Full-Stack Engineer** project. Developed with Node.js and Express, with product and checkout routes, validations, and automatic documentation with Swagger.


## 🛠️ Estrutura do Projeto

```bash
ecoloop-back/
├── controller/
│   └── v1/
│       ├── checkout.js        # Checkout logic
│       └── products.js        # Product filtering logic
├── data/
│   └── stock.json             # Product stock file
├── middleware/
│   ├── buildRoutes.js         # Dynamic router building
│   ├── buildSwagger.js        # Automatic generation of Swagger documentation
│   └── validateRequest.js     # Request validation middleware
├── routes/
│   └── routesV1.js            # API route definition
├── swagger/
│   └── swagger_output.json    # File generated by Swagger with documentation
├── index.js                   # Main server file
```


## Installing

1. Clone the repository:
   ```bash
   git clone <GitHub Repository URL>
   cd ecoloop-back
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

## Execution

### Dev Mode:
```bash
npm run dev
```

### Prod Mode:
```bash
npm start
```


## API routes

- **GET** `/products` - Lists products with optional filters:
    - `minPrice` (float): Minimum price
    - `maxPrice` (float): Maximum price
    - `name` (string): Name or part of the name of the product

- **POST** `/checkout` - Performs the checkout:
    - `cart` (array): List of items in the cart
    - Example of body:
    ```json
    {
        "cart": [
            { "id": 1, "quantity": 2 },
            { "id": 2, "quantity": 1 }
        ]
    }
    ```
