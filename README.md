## Installing

1. Clone the repository:
   ```bash
   git clone <GitHub Repository URL>
   cd ecoloop-back
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

## Execution

### Dev Mode:
```bash
npm run dev
```

### Prod Mode:
```bash
npm start
```


## API routes

- **GET** `/products` - Lists products with optional filters:
    - `minPrice` (float): Minimum price
    - `maxPrice` (float): Maximum price
    - `name` (string): Name or part of the name of the product

- **POST** `/checkout` - Performs the checkout:
    - `cart` (array): List of items in the cart
    - Example of body:
    ```json
    {
        "cart": [
            { "id": 1, "quantity": 2 },
            { "id": 2, "quantity": 1 }
        ]
    }
    ```
